# Выполненные задания по дисциплине ООП за 2 курс в НГУ ИИР.
Но задачи с ФИТа.

## Установка *Google Test* через apt
```
sudo apt install libgtest-dev cmake
cd /usr/src/gtest
sudo cmake .
sudo make
sudo cp lib/*.a /usr/lib
```
